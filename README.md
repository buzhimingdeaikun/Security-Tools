# Security-Tools
## 📌 项目简介
本仓库记录了我学习 Python 网络安全开发的完整过程，包含 30 天从基础到实战的练习代码与安全工具。

## 🎯 目标
- 掌握 Python 基础语法与常用库
- 独立编写目录扫描、端口扫描、漏洞检测等安全工具
- 积累可写入简历的实战项目

## 📂 目录结构
Security-Tools/
├── Day1-IP-Tool/ # 变量与输入输出
├── Day2-HTTP-Checker/ # if/else 状态码判断
├── Day3-Dict-Reader/ # 循环与字典读取
├── ...
├── DirScanner/ # 目录扫描器（项目1）
├── PortScanner/ # 端口扫描器（项目2）
└── README.md

## 🛠️ 技术栈
- Python 3.8+
- requests
- threading
- socket
- 正则表达式
